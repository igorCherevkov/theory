1) Какие типы данных представлены в TS?

TypeScript - это суперсет JavaScript, что означает, что он расширяет язык JavaScript, добавляя 
статическую типизацию и другие возможности. Одной из основных особенностей TypeScript являются 
типы данных.

Number: числовое значение
String: строка
Boolean: логическое значение true или false
Array: массивы
Symbol
BigInt: Для представления целых чисел больше 2^53 - 1
Null и undefined: null и undefined как и в javascript
Void
Enum
Any: произвольный тип
Unknown: произвольный тип
Never
